# AI-loan-prediction-app

A Machine Learning based Loan Prediction System that predicts whether a loan application will be approved based on applicant details. The project uses a trained Scikit-learn model and a Flask web application for prediction.

## Features
- Predict loan approval using Machine Learning
- User-friendly Flask web interface
- Trained Scikit-learn model
- Fast and accurate predictions

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML/CSS

## Installation

```bash
git clone https://github.com/CoderPrakhar/Loan-Prediction-System.git
cd Loan-Prediction-System
pip install -r requirements.txt
python app.py
```

## Dataset
The model is trained on a loan approval dataset using supervised machine learning techniques.

## Output
The application predicts whether a loan is likely to be **Approved** or **Rejected** based on user inputs.
